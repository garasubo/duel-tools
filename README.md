# duel-tools

遊戯王のための便利ツールをまとめたウェブアプリ

## 主な機能

### 戦績記録

- 対戦結果、先攻・後攻、自分のデッキ、相手デッキ、スコア、メモ、タグを記録できます。
- 記録した戦績は履歴画面で一覧表示し、条件で絞り込みながら確認できます。
- 履歴の詳細画面から、過去の対戦内容を編集・削除できます。
- 戦績データはCSVとしてエクスポート・インポートできます。
- デッキ名やタグは管理画面で追加、編集、削除できます。
- 全体勝率、先攻・後攻別勝率、デッキ別勝率を統計画面で確認できます。
- 自分のデッキと相手デッキの組み合わせごとの勝率を集計できます。
- デュエリストカップのDP推移をグラフで確認できます。
- 画面キャプチャとOCRにより、勝敗や先攻・後攻の自動検出を補助します。
- オーバーレイ画面を開き、配信や録画向けに戦績情報を表示できます。

### 初動率計算

- デッキ枚数、カード枚数、初動条件を入力して初動率を計算できます。
- 複数の初動パターンを登録し、いずれかを満たす5枚初手の確率を集計できます。
- カードにラベルを付け、同じ役割を持つカード群をまとめて条件に指定できます。
